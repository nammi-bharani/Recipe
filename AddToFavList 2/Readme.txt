Add to Fav Dish List:

Randomly chosen from the web. This app lets you add a dish to a list that you want to cook or interested in cooking to a list. Once you have the list you can access the recipe whenever you want. Also when recipe shows
Up from the web, the user can edit the recipe before saving to the fav list.
